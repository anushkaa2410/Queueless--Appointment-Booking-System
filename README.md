# Queueless--Appointment-Booking-System
QueueLess is a full-stack appointment booking and queue management platform built using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript, featuring token-based scheduling, real-time availability tracking, and flexible appointment management.

## Overview

QueueLess is a full-stack web-based appointment booking and queue management platform designed to reduce waiting times and improve service efficiency in organizations such as hospitals, clinics, banks, salons, blood banks, and government offices.

The system allows users to book appointments online, receive token numbers, view real-time slot availability, and manage bookings. Service providers can create schedules, manage services, control token limits, and monitor appointments through a dedicated admin dashboard.

---

## Features

### User Features
- User Registration & Login
- Search Service Providers
- View Available Time Slots
- Book Appointments Online
- Automatic Token Generation
- View Booking Status
- Cancel Appointments
- Real-Time Availability Tracking
- My Bookings Dashboard

### Admin Features
- Admin Registration & Login
- Create Weekly Schedules
- Manage Multiple Services
- Set Token Limits
- Mark Days as Closed
- Date-Based Schedule Overrides
- Monitor Bookings
- View Dashboard Analytics
- Manage Appointment Availability
